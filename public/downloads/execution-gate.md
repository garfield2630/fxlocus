# 执行闸门（Execution Gate / Pre-trade Gate）

> 目的：把“下单”变成“条件通过后的表达”。写不出证伪，就不要交易。

## 1) 市场与环境

- 品种（Symbol）：
- 周期（Timeframe）：
- 市场阶段（Regime）：趋势 / 震荡 / 转折
- 波动环境（Volatility）：低 / 中 / 高
- 时段（Session）：亚洲 / 欧洲 / 美盘

## 2) 位置（Location）

- 关键位置类型：分配位 / 防守位 / 反转位
- 结构边界（Boundary）：
- 你错在哪里（Where I’m wrong）：

## 3) 力量（Force）

- 推进质量（0-5）：
- 回撤性质（有序/无序）：
- 是否出现衰竭信号：

## 4) 证伪（Falsification）

用一句话写清楚：

- **如果发生 X，我立即退出**：
- 退出动作（Exit action）：

## 5) 风险（Risk）

- 单笔风险（%）：
- 当日风险上限（%）：
- 连续亏损状态：否 / 2 连亏 / 3 连亏（触发协议）

## 6) 结论

- Gate 结果：PASS / FAIL
- FAIL 时的动作：停止交易 / 降频 / 只复盘 / 关闭平台
