# 波动环境分层表（Volatility Tiers）

> 目的：让参数随波动变化，而不是用同一套参数跑所有市场。

## 1) 分层标准（示例）

- 低波动：ATR / 日均波幅 < A
- 中波动：A ≤ ATR / 日均波幅 < B
- 高波动：ATR / 日均波幅 ≥ B

> 你可以用：ATR、区间波幅、或者你习惯的波动指标。

## 2) 参数规则

| 波动层级 | 止损宽度 | 仓位 | 持仓时间 | 备注 |
|---|---|---|---|---|
| 低 | 较窄 | 正常 | 正常 | 更依赖位置过滤 |
| 中 | 标准 | 正常/减仓 | 正常 | 用执行闸门控制冲动 |
| 高 | 更宽 | 减仓 | 更短 | 更严格的 Fail 规则 |
